Shell Fonts
===========

A number of nice shell / editor fonts. From [programmingfonts](http://app.programmingfonts.org/)
and random exploration.

Recommended font size to the right of the name. For Macvim, subtract one from the size.

- DejaVu Mono 12
- Fira Mono 12
- Hasklig 12
- Inconsolata 13
- Input (Light) 12
- Liberation Mono 12
- Menlo 12
- Monaco 12
- Meslo 12
- **Office Code Pro 12**
- PT Mono 12
- Roboto Mono 12
- Source Code Pro 12
